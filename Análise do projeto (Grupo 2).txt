                                                    DIAGRAMA DE CLASSES

* A OBSERVAR

- SERIA MAIS APROPRIADO NA QUESTÃO CADASTRAL CRIAR UMA TABELA "PESSOA" PARA QUE DENTRO DELA FOSSEM ADICIONADOS OS ATRIBUTOS
NECESSÁRIOS PARA ESTE TIPO DE SITUAÇÃO (nome, endereço, etc.) E AS CLASSES (CLIENTE, VENDEDOR E GERENTE) HERDAREM DESSA
NOVA CLASSE "PESSOA";

- FALTAM ATRIBUTOS DENTRO DAS CLASSES: ENCOMENDA E VENDEDOR. ISSO SE DÁ PELO FATO DE HAVER POSSIBILIDADE DE MAIS DE UM VENDEDOR
OU MAIS DE UMA ENCOMENDA; 

* PONTOS POSITIVOS

- DIAGRAMA BEM ESTRUTURADO, DE FÁCIL COMPRRENSÃO COM AS CLASSES SENDO BEM CONECTADAS ENTRE SI.
